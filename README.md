# ბუნებრივი კატასტროფების ადრეული გაფრთხილების სისტემა

საბაკალავრო პროექტი - კავკასიის უნივერსიტეტი

## პროექტის აღწერა

ბუნებრივი კატასტროფების ადრეული გაფრთხილების სისტემა წარმოადგენს ვებ-აპლიკაციას, რომელიც მომხმარებლებს აწვდის დროულ შეტყობინებებსა და უსაფრთხოების რეკომენდაციებს მათი არჩეული მდებარეობის საფუძველზე.

სისტემა რეალურ დროში იღებს ამინდის მონაცემებს OpenWeather API-დან, ახდენს ამ მონაცემების ანალიზს წინასწარ განსაზღვრული წესების მიხედვით და განსაზღვრავს მოსალოდნელ რისკებს.

მომხმარებლებს შეუძლიათ რეგისტრაცია, ავტორიზაცია, რეგიონისა და ქალაქის არჩევა, შეტყობინებების მიღება და სხვადასხვა ტიპის ბუნებრივი კატასტროფისთვის განკუთვნილი უსაფრთხოების რეკომენდაციების გაცნობა.

# გამოყენებული ტექნოლოგიები

## Frontend

### React.js

გამოყენებულია მომხმარებლის ინტერფეისის (UI) შესაქმნელად. React უზრუნველყოფს კომპონენტურ არქიტექტურას, რაც ამარტივებს კოდის ორგანიზებასა და ხელახლა გამოყენებას.

### React Router

გამოყენებულია აპლიკაციის შიდა ნავიგაციისთვის. უზრუნველყოფს გვერდებს შორის სწრაფ გადასვლას ბრაუზერის სრული გადატვირთვის გარეშე.

### Axios

გამოყენებულია Backend API-სთან კომუნიკაციისთვის. მისი საშუალებით იგზავნება HTTP მოთხოვნები და მიიღება მონაცემები სერვერიდან.

### CSS3

გამოყენებულია აპლიკაციის ვიზუალური დიზაინისა და გვერდების სტილიზაციისთვის.

### React Icons

გამოყენებულია სხვადასხვა ვიზუალური აიქონების დასამატებლად, რაც აუმჯობესებს მომხმარებლის გამოცდილებას და ინტერფეისის აღქმას.

### Vite

გამოყენებულია Frontend პროექტის შესაქმნელად და გასაშვებად. უზრუნველყოფს სწრაფ კომპილაციასა და განვითარების პროცესს.

## Backend

### Python 3.10+

პროექტის Backend ნაწილის ძირითადი პროგრამირების ენა. არჩეულია სიმარტივის, ფართო ბიბლიოთეკებისა და ვებ-განვითარების მხარდაჭერის გამო.

### FastAPI

გამოყენებულია REST API-ის შესაქმნელად. უზრუნველყოფს სწრაფ, თანამედროვე და მარტივად დასადოკუმენტებელ Backend სისტემას.

### SQLAlchemy

გამოყენებულია მონაცემთა ბაზასთან სამუშაოდ (ORM). საშუალებას იძლევა მონაცემთა ბაზის ცხრილები წარმოდგენილი იყოს Python კლასების სახით.

### Uvicorn

გამოყენებულია FastAPI აპლიკაციის გასაშვებად. წარმოადგენს ASGI სერვერს, რომელიც ამუშავებს API მოთხოვნებს.

### JWT Authentication

გამოყენებულია მომხმარებლების ავტორიზაციისა და სესიის მართვისთვის. JWT ტოკენები უზრუნველყოფს დაცულ წვდომას ავტორიზებულ რესურსებზე.

### Passlib

გამოყენებულია პაროლების უსაფრთხოდ დასაშიფრად და შესამოწმებლად, რათა მომხმარებლის მონაცემები დაცული იყოს.

## მონაცემთა ბაზა

### SQLite

გამოყენებულია პროექტის მონაცემთა ბაზად. ინახავს მომხმარებლების, მდებარეობებისა და გაფრთხილებების ინფორმაციას. არჩეულია მისი სიმარტივისა და დამატებითი სერვერის არქონის გამო.

## გარე სერვისები

### OpenWeather API

გამოყენებულია რეალურ დროში ამინდის მონაცემების მისაღებად. მიღებული ინფორმაცია გამოიყენება რისკების ანალიზისა და ბუნებრივი კატასტროფების აღმოჩენისთვის.

## გამოყენებული ხელსაწყოები

### GitHub

გამოყენებულია ვერსიების კონტროლისა და პროექტის კოდის შესანახად. უზრუნველყოფს ცვლილებების ისტორიის მართვას და გუნდურ მუშაობას.

### PyCharm

გამოყენებულია Backend ნაწილის განვითარებისათვის. უზრუნველყოფს Python პროექტების მართვისა და გამართვის (debugging) შესაძლებლობებს.

### Postman

გამოყენებულია API Endpoint-ების ტესტირებისთვის და Backend ფუნქციონალის შესამოწმებლად.

### Swagger UI

გამოყენებულია FastAPI-ის მიერ გენერირებული API დოკუმენტაციისა და Endpoint-ების ინტერაქტიული ტესტირებისათვის.

# სისტემური მოთხოვნები

პროექტის გასაშვებად აუცილებელია:

- Python 3.12 ან უფრო ახალი ვერსია
- Node.js (LTS Version)
- npm (Node Package Manager)
- Git
- OpenWeather API Key

# ინსტალაცია

## Backend-ის გაშვება

გადადით backend საქაღალდეში:

```bash
cd backend
```

დააინსტალირეთ საჭირო ბიბლიოთეკები:

```bash
pip install -r requirements.txt
```
გაუშვით სერვერი:

```bash
uvicorn app.main:app --reload
```

Backend ხელმისაწვდომი იქნება მისამართზე:

```text
http://127.0.0.1:8000
```

Swagger დოკუმენტაცია:

```text
http://127.0.0.1:8000/docs
```

---
## Node.js-ის ინსტალაცია

Frontend ნაწილის გასაშვებად აუცილებელია Node.js-ისა და npm-ის ინსტალაცია.

Node.js-ის გადმოწერა შესაძლებელია ოფიციალური ვებგვერდიდან:

https://nodejs.org

ინსტალაციის დასრულების შემდეგ შეამოწმეთ:

```bash
node -v
npm -v
```
## Frontend-ის გაშვება
გადადით frontend საქაღალდეში:

```bash
cd frontend
```

დააინსტალირეთ დამოკიდებულებები:

```bash
npm install
```

გაუშვით პროექტი:

```bash
npm run dev
```

Frontend ხელმისაწვდომი იქნება მისამართზე:

```text
http://localhost:5173
```
